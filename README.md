## pySim - Read, Write and Browse Programmable SIM/USIM Cards
====================================================
Installation
------------

Please install the following dependencies:

 - pyscard
 - serial
 - pytlv
 - cmd2 >= 1.3.0 but < 2.0.0
 - jsonpath-ng
 - construct
 - bidict
 - gsm0338

###  Install
yay -Sy python-pysim-git

### Uninstall
sudo pacman -Rs python-pysim-git
```
